# Student Management System  

**Full-Stack Web Application**  

A simple **Student Management System** that allows you to **Create, Read, Update, and Delete (CRUD)** student records, with **Admin and Student login functionality**.  

Frontend: **HTML, CSS, JavaScript, Bootstrap 5, AI**  
Backend: **Java Spring Boot, MySQL**
